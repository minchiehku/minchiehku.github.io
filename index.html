Paper

Research on Scheduling and Allocation for Unmanned Taxi Dispatch Services.

  • Through designing a ride-matching algorithm in a Java-based simulation environment, a ride-matching system for unmanned taxis was developed.
  • Using object-oriented design, each role is abstracted as an independent object, calculating the next event's occurrence time and storing it in an ArrayList as an event queue. In each iteration, the soonest event time is selected to advance system time, while triggering movements between objects and calculating new event times after movement.
  • The algorithm focuses on the matching problem of multiple customers and multiple vehicles, with each vehicle being paired with only one customer. Under limited resources and time, only approximate optimal solutions can be found. This problem belongs to the highly complex NP-hard problems.
  • The algorithm considers the customers' tolerable waiting times and different vehicle dispatch methods, reducing the overall average waiting time and the worst-case waiting time.
